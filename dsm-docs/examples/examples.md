# Package Examples

## Overview

This page is a navigation hub for Synology package development examples within the developer guide documentation.

## Available Examples

The page lists four primary package examples:

1. **Open Source Tool: tmux** - Demonstrates packaging an open-source terminal multiplexer
2. **Open Source Tool: nmap** - Shows how to package the network scanning utility
3. **Docker package** - Illustrates Docker package creation
4. **Web Package: WordPress** - Example of packaging a web application

- Getting started and system requirements
