# Get Started with Publishing

## Overview

This page provides guidance for developers interested in publishing packages to Synology Package Center.

## Publishing Requirements

The publishing process involves these key steps:

1. **Developer Application**: Submit an application on the Synology website at the developer support portal
2. **Agreement Acceptance**: Review and accept both the Developer Distribution Agreement and Package Developer Guideline
3. **Compliance**: Ensure packages adhere to Package Center's Terms of Service

## Quality Emphasis

Package quality directly influences the long-term success of your package in terms of installation, online reviews, engagement, and user retention.
