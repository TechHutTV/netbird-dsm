# Application I18N

## Directory Structure

```
ui (specified by dsmuidir in INFO)
в””в”Җв”Җ texts
   в”ңв”Җв”Җ enu
   в”Ӯ    в””в”Җв”Җ strings
   в””в”Җв”Җ cht
        в””в”Җв”Җ strings
```

## Strings File Format

### English (`texts/enu/strings`)

```ini
[app_tree]
index_title="This is a title"
node_1="This is node1"
[app_tab]
tab1="This is tab1"
tab2="This is tab2"
```

### Traditional Chinese (`texts/cht/strings`)

```ini
[app_tree]
index_title="йҖҷжҳҜжЁҷйЎҢ"
node_1="йҖҷжҳҜзҜҖй»һ1"
```

## Usage

Reference i18n strings as `section:key`:

```
"title": "app_tree:node_1"
```

One value can only be one i18n string.

## Important

I18N strings load only when applications open on desktop after DSM 7.0. For desktop notifications, include strings in the `preloadTexts` field of the application config.
